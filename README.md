-------------------------------

使用方法

直接双击syoFunction.exe

输入小说链接

例如：https://ncode.syosetu.com/n4002gy/

输入端口号（选项）

如果不填端口号，则默认直接访问

点击OK

------------------------------
日本語の使用方法

syoFunction.exeを開く

小説のURL　linkを入力ください　　*URL　link最後の「/」は削除できません

Https　Proxyを使っている場合はportを入力してください

プログラムは自動的に

http://localhost:port　

のように追加する

portは入力されない場合は直接URLアクセスする
